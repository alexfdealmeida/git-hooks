# git-hooks

Este repositório armazena os **hooks** para padronizar e realizar validações nos commits. 
> Para ativar os hooks, deve-se adicionar o git-hooks como **submodule** e copiar os arquivos da pasta `/initialize-submodule` para a raiz do repositório.

## Como obter uma cópia

Caso deseje obter uma cópia local:

- **GSK (recomendado)**

  O GSK (Git Script Kit) automatiza a clonagem e configurações básicas.

  ```sh
  gsk clone https://github.com/alexfdealmeida/git-hooks.git
  ```

- Git (manual)

  ```sh
  git clone https://github.com/alexfdealmeida/git-hooks.git
  cd git-hooks && ./getting-started.sh
  ```
